# 1.数据类型

# 1.1基本类型

整数：int
浮点数：float
Number:数字
bool：布尔类型，表示真，假
complex：复数

# 1.2类型查看

1.type(数字或变量)作用是查看类型
2.type(2/2)得到是float类型，“/”是除法，得到的是float类型
3.type(2//2)得到是int类型，“//”是整除，得到的是int类型

# 1.3其他任何类型以bool类型输出

bool(1)输出：true          bool(0)输出：false

bool("abc")输出：true      bool("")输出：false

boo([1,2])输出：true	   bool([])输出：false

bool({1,1})输出：true      bool({})输出：false

bool(None)输出：true
