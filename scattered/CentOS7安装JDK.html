# CentOS7安装JDK

# 1.服务器安装JDK之前，先确保有没有默认安装JDK，执行以下命令检查

java -version

如果是自己在虚拟机上安装的centos7.x或者centos6.x，这会默认帮你安装好OpenJDK，则需要卸载，卸载步骤如下1.1~1.2所示

# 1.1检查系统安装的OpenJDK

rpm -qa|grep openjdk -i

# 1.2删除搜索出来的openjdk的安装包并且检查后显示删除完毕

rpm -e --nodeps 需要删除的软件名称

例如：rpm -e --nodeps java-1.8.0-openjdk-headless-1.8.0.181-7.b13.e17.x86_64

# 2.创建jdk安装目录以及上传jdk压缩包解压

# 2.1创建jdk安装目录

mkdir /usr/java

# 2.2创建上传压缩包目录，以及上传jdk压缩包

mkdir /home/software

scp 本地jdk压缩包 root@ip:/home/software

# 2.3解压jdk压缩包

tar -zxvf jdk-8u191-linux-x64.tar.gz

# 3.配置jdk环境变量

# 3.1将解压文件移动到指定目录

mv jdk-8u191-linux-x64.tar.gz /usr/java

# 3.2修改系统文件

vim /etc/profile

# 3.3移动到文件最底部，添加如下内容

export JAVA_HOME=/usr/java/jdk1.8.0_11
export CLASSPATH=.:%JAVA_HOME%/lib/dt.jar:%JAVA_HOME%/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin

# 3.4刷新profile，时期生效

source /etc/profile

# 4.检查JDK环境

java -version

至此，JDK安装成功
