# Redis的命令行客户端

  • 关闭redis

    redis-cli -a password shutdown

  • 关闭redis

    cd /etc/init.d/

./redis_init_script stop

  • 进入redis客户端

    redis-cli

  • 输入密码

    auth pwd

  • 设置缓存

    set key value

  • 获得缓存

    get key

  • 删除缓存

    del key

  • 查看redis是否存活

    redis-cli -a password ping
